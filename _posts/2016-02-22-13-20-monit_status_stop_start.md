Obtaining a list of monitorings:

{% highlight bash %}
monit status
{% endhighlight %}

Stopping a monitor:
{% highlight bash %}
monit stop name_of_monitor
{% endhighlight %}

Starting a monitor:
{% highlight bash %}
monit start name_of_monitor
{% endhighlight %}
